Detailed Description

CaptureBoard The capture board is an exections of the playing board for each robot in order to keep track and place the disgarded peices.

You can think of it like a stack that you push and pop pecies on and off of. Structurally it is just a 2D array in which the 1st half of the rows represent one half of the board and the second. Running the test cases should give a pretty clear demenstration of how the data structure works
